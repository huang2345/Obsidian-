Attr接口表示Element节点中的属性

| 属性           | 描述                         |
| ------------ | -------------------------- |
| isId         | 若该节点表示文档中的某个ID，则为true      |
| name         | 属性名                        |
| value        | 属性值                        |
| ownerElement | 拥有该属性节点的节点，null表示该属性节点未被使用 |
| specified    | 若该节点的属性值由文档设置，则为true       |
