
| 转义字符   | 说明      |
| ------ | ------- |
| \b     | 退格      |
| \n     | 回车      |
| \t     | Tab     |
| \f     | 换页      |
| \\'    | 单引号     |
| \\"    | 双引号     |
| \v     | 跳格(Tab) |
| \r     | 换行      |
| \\\    | \\      |
| \OOO   | 八进制整数   |
| \xHH   | 十六进制整数  |
| \uhhhh | 十六进制编码  |
