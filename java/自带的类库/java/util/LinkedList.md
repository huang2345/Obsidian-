# LinkedList\<E> implements List
类
### 构造器
#### LinkedList()
构造一个空链表
#### LinkedList(Collection\<? extends E> elements)
构造一个链表，并将集合中的所有元素添加到链表中
### 实例方法
#### void addFirst(E element)
#### void Last(E element)
将元素添加到表头或表尾
#### E getFirst()
#### E getLast()
获取表头或表尾的元素
#### E removeFirst()
#### E removeLast()
删除并返回表头或表尾的元素。
